🚀 Quick Start

Three commands.

📦 1. Pip Install
$ pip install ivllm-hust
🔧 2. Setup Backend
$ vllm-hust install cuda # or ascend / cpu
⚡ 3. Run Inference
$ vllm-hust run -p "Hello AI" --backend cuda
💡 Note: Pick the backend that matches your hardware.
