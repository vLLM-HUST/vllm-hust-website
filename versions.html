🚀 vllm-hust

Upstream-compatible vLLM packages for domestic hardware and AGI4S serving

📥 Installation Entry

# Install the default serving stack
pip install vllm-hust

# Or install from PyPI with an explicit version
pip install vllm-hust==latest

vllm-hust 是由 HUST 维护的 vLLM fork 发行包。当前页面展示的是实际已发布包版本； 对外稳定发布建议采用“上游基线 + HUST 修订号”的规则，例如 0.17.2.post1。 这样既能说明它基于哪个上游版本，也能避免与官方同号版本产生等价误解。
