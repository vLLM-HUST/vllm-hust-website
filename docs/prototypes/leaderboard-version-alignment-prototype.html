Leaderboard Version Alignment Prototype

这个原型只演示版本区视觉结构，不改生产页面。目标是让引擎名左对齐、版本号列对齐，并把第三行日期做成整体左对齐，避免继续挤压右侧操作按钮。

Visual Rules

  • `vllm-hust` 与 `vllm-ascend-hust` 分两行展示，左列标签对齐，右列版本号对齐。
  • 如果存在 `postN` 则保留；如果没有 `postN`，直接忽略，不展示 `devNNN`。
  • `vllm` 与 `vllm-ascend` 采用同一套两列规则，保证榜单扫读一致。
  • 第三行日期保持左对齐，不再缩进到版本列下方。
